# **App Name**: WellTrack

## Core Features:

- User Authentication & Profile: Secure registration/login with email and social media. Customizable user profiles.
- Habit Tracking: Log and visualize progress on daily habits, and visualize in multiple ways such as a calendar heatmap.
- Mood Tracking: Daily mood logging with emoji scales. Generate weekly/monthly mood pattern analytics with charts.
- AI Insights Engine: Machine learning provides personalized insights to improve mental health and form positive habits; this tool is essential to provide feedback on how well the user is improving and what they need to work on to make improvements.
- Smart Notifications: Reminders to encourage the user to keep on track and stay focused throughout the day.
- Gamification & Motivation System: Points, badges, and levels provide feedback, and motivation to improve.
- Goal setting: Users can create SMART goals in the app to drive them towards improvements.

## Style Guidelines:

- Primary color: Light desaturated green (#90EE90) to represent growth, health, and tranquility. Primary green is one of the accent colors on the flag for mental health awareness.
- Background color: Very light desaturated green (#F0FFF0). This color will subtly invoke green from the primary color without overpowering.
- Accent color: Light yellow (#FAFAD2), used for highlighting interactive elements such as buttons or selected menu items. Represents cheerfulness, which encourages users on their road to wellness.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for headlines or body text
- Use simple, clean icons throughout the app for easy navigation.
- Subtle transitions and animations to guide the user through the app and provide feedback on actions.
